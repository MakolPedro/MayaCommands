# Set Log Channel
> Escolha um canal para receber todos os log(relatórios) do servidores.
> O Log Channel é de grande ajuda para a organização. Todo mute, kick, ban vai para o log channel, detalhadamente com todos os ocorridos.

### Comando
`-setlogchannel #canal`
